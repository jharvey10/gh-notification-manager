After any code changes are made, the following requirements must be met before the work can be
considered complete:

1. All code must pass linting (`npm run lint`), formatting (`npm run format`), and typechecking
   (`npm run typecheck`).
2. All SonarQube issues must be resolved or strong justification for leaving them must be provided.
3. No existing code comments shall be deleted unless they are incorrect, in which case they should
   be updated, not removed.

For all of the above, it does not matter if issues were pre-existing or not. They must be fixed.
